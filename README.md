# Omar Hafez Portfolio v4

Personal portfolio website built with HTML, CSS, and JavaScript.

This version keeps the terminal-inspired design and personal voice of v3. The main upgrade is not a new visual identity. It is clearer project writing, stronger project hierarchy, and a sharper explanation of the thinking behind my data work.

## What Changed in v4

- Kept the existing terminal-style visual language, theme toggle, typing effect, skill filters, and project cards
- Kept the hero and terminal panel centered on data analysis, SQL, and Python
- Kept the flagship data projects separate from smaller web and game projects
- Added an “Other Projects” section for Vertex School Website and Etherea: Blood Wing
- Added a gameplay preview for Etherea: Blood Wing, a Python and Tkinter dark fantasy RPG prototype inspired by my Minecraft Bedrock project
- Kept the strongest data projects front and center

## Features

- Responsive layout for desktop and mobile
- Light/dark theme preference stored in the browser
- Keyboard-visible focus states, skip link, semantic sections, and reduced-motion support
- Mobile navigation, active-section highlighting, skill filtering, and copy-email interaction
- Open Graph metadata and local assets for share previews

## Run Locally

Open `index.html` in a browser, or use a simple local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This is a static website and is ready for GitHub Pages. Push the project files to the repository used for `hafezomar.github.io`, then enable GitHub Pages in the repository settings if needed.
